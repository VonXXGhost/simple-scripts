### potplayer_auto_save_episode.py
仅windows用。基于win32gui接口的窗口标题文字识别，以存档文件与窗口标题的差异度作为判断基准。使用前需要更改anime_path为自己的动画存放文件夹，ex_dirs为不需要存档的文件夹。推荐改为pyw后缀以无窗口模式使用。
